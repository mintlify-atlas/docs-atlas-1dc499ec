The `pareto` module provides 80/20 (Pareto) analysis functionality to identify high-value customers and products that drive the majority of sales.

---

## Core Function

### calcular_pareto

```python
def calcular_pareto(
    df: pd.DataFrame,
    columna_valor: str,
    columna_id: str,
) -> pd.DataFrame
```

Adds Pareto analysis columns to a DataFrame.

**Parameters:**

<ParamField path="df" type="pd.DataFrame" required>
  Input DataFrame (typically aggregated by customer or product)
</ParamField>

<ParamField path="columna_valor" type="str" required>
  Column name containing the metric to analyze (e.g., `"importe_total"`)
</ParamField>

<ParamField path="columna_id" type="str" required>
  Column name with the entity ID (e.g., `"id_cliente"`, `"id_producto"`)
</ParamField>

**Returns:**

DataFrame sorted descending by `columna_valor` with these **added columns**:

<ParamField path="participacion_pct" type="float">
  Individual share of total (0-1). Example: 0.15 = 15% of total sales.
</ParamField>

<ParamField path="acumulado_pct" type="float">
  Cumulative share (0-1). Example: 0.78 = this entity + all above = 78% of total.
</ParamField>

<ParamField path="categoria_pareto" type="str">
  - **"A"**: Top entities (0-80% cumulative)
  - **"B"**: Middle entities (80-95% cumulative)
  - **"C"**: Long tail (95-100% cumulative)
</ParamField>

**Usage Example:**

```python
from src.pareto import calcular_pareto
import pandas as pd

# Customer sales data
df_clientes = pd.DataFrame({
    "id_cliente": ["C001", "C002", "C003", "C004", "C005"],
    "importe_total": [50000, 30000, 15000, 3000, 2000]
})

df_pareto = calcular_pareto(df_clientes, "importe_total", "id_cliente")

print(df_pareto)
```

**Output:**

```
  id_cliente  importe_total  participacion_pct  acumulado_pct categoria_pareto
0      C001          50000              0.50           0.50               A
1      C002          30000              0.30           0.80               A
2      C003          15000              0.15           0.95               B
3      C004           3000              0.03           0.98               C
4      C005           2000              0.02           1.00               C
```

<Note>
  The function automatically sorts the DataFrame by `columna_valor` in descending order before calculating cumulative percentages.
</Note>

## Concentration Metrics

### get_concentracion_pareto

```python
def get_concentracion_pareto(
    df: pd.DataFrame,
    columna_valor: str = "importe_total",
) -> dict
```

Calculates concentration metrics for a Pareto-analyzed DataFrame.

**Parameters:**

<ParamField path="df" type="pd.DataFrame" required>
  DataFrame with `categoria_pareto` column (output from `calcular_pareto()`)
</ParamField>

<ParamField path="columna_valor" type="str">
  Value column name (default: `"importe_total"`)
</ParamField>

**Returns:**

Dictionary with concentration metrics:

```python
{
    "pct_entidades_A": float,   # % of entities in category A (0-1)
    "pct_ventas_A": float,      # % of sales from category A (0-1)
    "n_entidades_A": int,       # Count of A entities
    "indice_gini": float        # Gini coefficient (0=equality, 1=total concentration)
}
```

**Usage Example:**

```python
from src.pareto import calcular_pareto, get_concentracion_pareto

df_pareto = calcular_pareto(df_clientes, "importe_total", "id_cliente")
concentracion = get_concentracion_pareto(df_pareto)

print(f"Top {concentracion['pct_entidades_A']*100:.0f}% of customers")
print(f"represent {concentracion['pct_ventas_A']*100:.0f}% of sales")
print(f"Gini coefficient: {concentracion['indice_gini']:.3f}")
```

**Sample Output:**

```
Top 15% of customers
represent 78% of sales
Gini coefficient: 0.542
```

**Gini Coefficient Interpretation:**

- **0.0 - 0.3**: Low concentration (sales evenly distributed)
- **0.3 - 0.5**: Moderate concentration (typical B2B distribution)
- **0.5 - 0.7**: High concentration (few customers dominate)
- **0.7 - 1.0**: Extreme concentration (risk: dependency on few accounts)

## Internal Functions

### _calcular_gini

```python
def _calcular_gini(valores: np.ndarray) -> float
```

Calculates the Gini coefficient for an array of values.

**Parameters:**

<ParamField path="valores" type="np.ndarray" required>
  Numeric array (e.g., sales amounts)
</ParamField>

**Returns:**

`float` between 0 and 1:
- **0**: Perfect equality (all values are the same)
- **1**: Perfect inequality (one value has everything)

**Formula:**

```
Gini = (2 * Σ(i * x[i])) / (n * Σ(x[i])) - (n + 1) / n
```

where values are sorted ascending and `i` is the rank.

## Real-World Usage

### In kpi_engine.py (Product Analysis)

```python
from src.pareto import calcular_pareto

def kpi_productos(df: pd.DataFrame, df_anterior: pd.DataFrame | None = None) -> pd.DataFrame:
    grp = df.groupby("id_producto").agg(
        importe_total=("importe_neto", "sum"),
        unidades_vendidas=("cantidad", "sum"),
    ).reset_index()

    # Add Pareto columns
    grp = calcular_pareto(grp, "importe_total", "id_producto")

    return grp
```

### In kpi_engine.py (Customer Analysis)

```python
def kpi_clientes(df: pd.DataFrame, clientes_df: pd.DataFrame, ...) -> pd.DataFrame:
    grp = df.groupby("id_cliente").agg(
        importe_total=("importe_neto", "sum"),
        frecuencia_compra=("id_venta", "count"),
    ).reset_index()

    grp = calcular_pareto(grp, "importe_total", "id_cliente")

    # Flag high-value customers at risk
    grp["flag_en_riesgo"] = (
        (grp["categoria_pareto"].isin(["A", "B"])) &
        (grp["dias_desde_ultima_compra"] > 30)
    )

    return grp
```

## Edge Cases

<Note>
  **Empty DataFrame:** If `df` is empty or `columna_valor` not found, the function returns `df` with default columns:
  - `participacion_pct = 0.0`
  - `acumulado_pct = 0.0`
  - `categoria_pareto = "C"`
</Note>

<Note>
  **Zero Total:** If sum of `columna_valor` is zero, all participation percentages are 0 and all entities are categorized as "C".
</Note>

---

## Performance Considerations

- **Sorting**: The function sorts the entire DataFrame by `columna_valor` (descending). For large datasets (>100k rows), consider pre-aggregating.
- **Memory**: Creates a copy of the input DataFrame. Original is not modified.
- **Gini Calculation**: O(n) complexity. Efficient for datasets up to millions of rows.
